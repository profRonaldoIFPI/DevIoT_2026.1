**🌐 Repositório: Desenvolvimento para Internet das Coisas (IoT) - 2026.1**

**Bem-vindo(a) ao repositório oficial da disciplina de Dev IoT** do curso de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) do Instituto Federal do Piauí (IFPI) - Campus Floriano, sob orientação do Prof. Ronaldo Borges.

Este espaço foi criado para armazenar, versionar e compartilhar todos os projetos e experimentos desenvolvidos durante os nossos encontros semanais. O objetivo é que este repositório sirva como um acervo da turma e um guia prático de consulta rápida para apoiar a construção das atividades avaliativas e dos projetos finais.
